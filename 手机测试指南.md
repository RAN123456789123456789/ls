# 手机测试小程序指南

## 📱 小程序信息

- **小程序名称**：图书借阅
- **AppID**：wxc514634bfc130f84

## 🚀 方法一：真机调试（最快，推荐开发时使用）

### 步骤：

1. **打开微信开发者工具**
   - 确保项目已打开并编译成功

2. **点击"真机调试"按钮**
   - 在工具栏找到"真机调试"图标（手机图标）
   - 或使用快捷键（查看工具菜单）

3. **扫码连接**
   - 工具会显示一个二维码
   - 用手机微信扫描二维码
   - 手机自动打开小程序

4. **开始调试**
   - 手机上的操作会同步到开发者工具
   - 可以在工具中查看日志和调试信息

### ⚠️ 注意事项：
- 真机调试需要手机和电脑在同一网络（WiFi）
- 每次重新编译后需要重新扫码
- 调试会话有时间限制

## 📤 方法二：上传体验版（可长期使用）

### 步骤：

#### 1. 上传代码

1. **在微信开发者工具中**
   - 点击工具栏的 **"上传"** 按钮
   - 填写版本号（如：`1.0.0`）
   - 填写项目备注（如：`测试版本`）
   - 点击 **"上传"**

2. **等待上传完成**
   - 上传成功后会有提示

#### 2. 设置为体验版

1. **登录微信公众平台**
   - 访问：https://mp.weixin.qq.com/
   - 使用管理员账号登录

2. **进入版本管理**
   - 左侧菜单：**开发** → **版本管理**
   - 找到 **"开发版本"** 标签页

3. **选为体验版**
   - 找到刚上传的版本
   - 点击 **"选为体验版"** 按钮
   - 确认操作

#### 3. 添加体验者（如果需要）

1. **进入成员管理**
   - 左侧菜单：**设置** → **成员管理**
   - 点击 **"体验者"** 标签

2. **添加体验者**
   - 点击 **"添加"** 按钮
   - 输入要添加的微信号
   - 发送邀请

3. **体验者接受邀请**
   - 体验者在微信中接受邀请

#### 4. 在手机上搜索小程序

**方式1：搜索小程序名称**
- 打开微信
- 下拉显示搜索框
- 搜索：**"图书借阅"**
- 点击进入小程序

**方式2：搜索AppID**
- 打开微信
- 下拉显示搜索框
- 搜索：**"wxc514634bfc130f84"**
- 点击进入小程序

**方式3：通过小程序码**
- 在微信公众平台生成小程序码
- 用微信扫描小程序码

## 🎯 方法三：发布正式版（所有人可搜索）

### 步骤：

1. **上传代码**（同方法二步骤1）

2. **提交审核**
   - 在微信公众平台 **版本管理** → **开发版本**
   - 点击 **"提交审核"**
   - 填写审核信息：
     - 服务类目
     - 标签
     - 功能页面
     - 等等

3. **等待审核**
   - 通常需要1-7个工作日
   - 可以在版本管理中查看审核状态

4. **发布上线**
   - 审核通过后，点击 **"发布"**
   - 小程序正式上线
   - 所有人可以在微信中搜索到

## 🔍 常见问题

### Q1: 真机调试扫码后提示"网络错误"
- **解决**：确保手机和电脑连接同一WiFi
- **解决**：检查防火墙设置

### Q2: 搜索不到小程序
- **检查**：是否已设置为体验版
- **检查**：是否添加了体验者（如果需要）
- **检查**：小程序名称是否正确

### Q3: 体验版无法访问
- **检查**：是否在体验者列表中
- **检查**：是否已接受体验邀请

### Q4: 上传代码失败
- **检查**：是否登录了正确的微信账号
- **检查**：是否有上传权限
- **检查**：网络连接是否正常
